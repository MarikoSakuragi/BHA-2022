**Settings**<br/>
| Standard　|　Metric　|
| Pressure Angle　|　20deg　|
| Module　|　1　|
| Number of Teeth　|　24　|
| Backlash　|　0mm　|

| Gear Thickness　|　2.5mm　|
| Pitch Diameter　|　2.5mm　|
<br/>
Standard: Unit of length<br/>
Pressure Angle: The size of the cutting edge<br/>
Module: Pitch Diameter/Number of Teeth<br/>
